# Security Policy

## Supported Versions

We will provide security updates for the latest version of Mesop.

## Reporting a Vulnerability

Please [report security vulnerabilities](https://github.com/mesop-dev/mesop/security/advisories/new) via GitHub's [security advisory](https://github.com/mesop-dev/mesop/security/advisories) feature. This allows you to collaborate with the Mesop maintainers privately while the vulnerability is getting fixed.
